## Installation
Although I train my model on Kaggle, I paste some command for needed packages
```
pip install numpy
pip install torch
pip install Pillow
pip install scikit-learn
pip install tqdm
pip install timm
pip install pandas
```
## Get Model & Predictions
Use `Code/Training.py` to get `best_model.pth`, and then make this model an input to run `Inference.py`.
This way, you can get final prediction.
